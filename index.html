INITIAL SETUP
REQUIRED ▼
STEP 01
Supabase プロジェクトを作成
supabase.com で無料アカウントを作成し、新しいプロジェクトを作成してください。

プロジェクト作成後、Settings → API から以下をメモします：
Project URL（https://xxxx.supabase.co）
anon public キー
STEP 02
テーブルを作成（SQL）
Supabase の SQL Editor を開き、以下のSQLを実行してください。
CREATE TABLE characters ( id TEXT PRIMARY KEY, data JSONB, created_at TIMESTAMPTZ DEFAULT NOW() ); CREATE TABLE expressions ( id TEXT PRIMARY KEY, char_id TEXT REFERENCES characters(id) ON DELETE CASCADE, label TEXT, category TEXT, image_path TEXT, created_at TIMESTAMPTZ DEFAULT NOW() ); CREATE TABLE npcs ( id TEXT PRIMARY KEY, data JSONB, created_at TIMESTAMPTZ DEFAULT NOW() ); CREATE TABLE npc_expressions ( id TEXT PRIMARY KEY, npc_id TEXT REFERENCES npcs(id) ON DELETE CASCADE, label TEXT, image_path TEXT, created_at TIMESTAMPTZ DEFAULT NOW() );
STEP 03
Storage バケットを作成
Supabase の Storage メニューから新しいバケットを作成します。

バケット名：images
Public bucket：ON（チェックを入れる）

次に Storage → Policies から以下のSQLを実行します：
-- images バケットへの読み書きを許可 CREATE POLICY "allow all" ON storage.objects FOR ALL USING (bucket_id = 'images') WITH CHECK (bucket_id = 'images');
STEP 04
URLとキーを入力して接続
探索者記録簿のページを開き、右上の ⚙ 設定 からProject URL と Anon Key を入力して「接続する」を押してください。

設定はブラウザに保存されるため、次回以降は自動接続されます。
⚙ DB設定を開く →
※ Supabase の無料プランで動作します（月500MB ストレージ・50,000行まで）。
※ データはお使いのSupabaseプロジェクトに保存されます。このツール自体はデータを収集しません。
